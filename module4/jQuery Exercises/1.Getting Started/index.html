Exercise 1: Getting Started

This exercise is about getting up and running. The point is to establish that jquery is installed and available for use. To that end, follow these steps:

  • Download jQuery and copy it into this exercise's folder.
  • Add the script tag into this page that points to the local jQuery file.
  • In the 'script.js' file, add code to either log a 'Hello, World!' message to the log in the Developer Tools, or to popup that message in an alert window.
  • Once you know javascript is working, output the jQuery object using either the log or alert methods.
